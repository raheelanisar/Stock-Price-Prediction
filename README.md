## 📌 Project Title
Stock Price Prediction using Machine Learning

---

## 📈 Project Overview
This project is a Machine Learning-based stock price prediction system that uses historical stock data to predict future closing prices. The project is deployed using Streamlit for interactive visualization.

---

## 🎯 Objective
To predict stock closing prices using features such as Open, High, Low, and Volume.

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- yFinance
- Streamlit
- Pickle

---

## 🤖 Model Used
- Linear Regression (Machine Learning)
- Regression Problem

---

## 📊 Features
- Real-time stock data fetching  
- Stock symbol input  
- Price prediction using ML model  
- Actual vs Predicted graph visualization  
- Simple Streamlit UI  

---

## 📌 Example Stock Symbols
- AAPL (Apple)  
- TSLA (Tesla)  
- GOOGL (Google)  
- MSFT (Microsoft)  

---

## 📊 Results
The model successfully predicts stock prices based on historical trends and shows comparison between actual and predicted values.

## Objective
Predict next day stock closing price using historical 
stock market data of Apple (AAPL).

## Dataset
- **Source:** Yahoo Finance (yfinance library)
- **Stock:** Apple Inc. (AAPL)
- **Period:** 2022-01-01 to 2024-01-01
- **Features:** Open, High, Low, Volume, Close

## Tech Stack
- Python, Pandas, NumPy
- Matplotlib
- Scikit-learn
- yfinance
- Jupyter Notebook

## Visualizations
- 📈 Actual vs Predicted Stock Price

## Model Used
- **Linear Regression**
- **R2 Score:** 0.99+
- **MAE:** Very Low

## Key Insights
- Apple stock data from 2022-2024 used
- Linear Regression gave excellent R2 Score
- Predicted price closely follows actual price
- No dataset download needed — yfinance fetches live data
